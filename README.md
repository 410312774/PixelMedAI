# PixelMedAI
## PixelMedAI是一款影像组学与深度学习任务平台，能够实现许多主流文献中研究方法，目前主要能实现  
1.影像组学任务 （生境影像组学，血管影像组学，传统影像组学，分形维影像组学，莫兰指数生境）  
 CT—ECV  
2.深度学习分类任务   
3.深度学习分割任务   
4.传统影像组学特征联合深度学习特征分类任务 
 功能更新日志：https://p.kdocs.cn/s/SROPMBAAIA
